# Dr. Lucas Duarte - Website Profissional

Website moderno e responsivo para o Dr. Lucas Duarte, médico recém-formado dedicado ao cuidado humanizado e promoção da saúde.

## Estrutura do Projeto

```
Dr.LucasDuarte/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos do site
│   ├── js/
│   │   └── main.js         # JavaScript interativo
│   └── images/             # Imagens do site
└── README.md
```

## Funcionalidades

- **Design Responsivo**: Adaptável para desktop, tablet e mobile
- **Menu Mobile**: Navegação otimizada para dispositivos móveis
- **Animações Suaves**: Efeitos de scroll e transições elegantes
- **Formulário de Contato**: Com validação e feedback visual
- **Botão WhatsApp**: Flutuante para contato rápido
- **SEO Otimizado**: Meta tags configuradas para melhor indexação

## Seções

1. **Hero**: Apresentação principal com chamada para ação
2. **Sobre**: Biografia e valores do médico
3. **Especialidades**: Áreas de atuação
4. **Formação**: Timeline acadêmica
5. **Conteúdo**: Espaço para artigos e publicações
6. **Contato**: Formulário e informações de contato

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com variáveis customizadas
- JavaScript ES6+ vanilla
- Google Fonts (Inter & Playfair Display)
- Font Awesome 6 (ícones)

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador
3. Para desenvolvimento, use um servidor local (Live Server, etc.)

## Personalização

### Cores
Edite as variáveis CSS no início do arquivo `style.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #0891b2;
    /* ... */
}
```

### Conteúdo
- Substitua os textos placeholder no `index.html`
- Adicione imagens reais na pasta `assets/images/`
- Atualize links de redes sociais e WhatsApp

### Imagens Recomendadas
- Foto profissional do médico (hero e sobre)
- Logo personalizado (opcional)
- Imagens para artigos do blog
